# 🔒 FocusLock

**FocusLock** is a smart parental control tool designed to help students stay focused during exams by limiting game and app usage.

---

## 🚀 Features

* ⏱️ Set daily time limit for apps
* 🎮 Track usage of selected apps
* 🔒 Auto-lock apps after time limit
* ⚠️ Warning message when time is over
* 💻 Simple and lightweight Python app

---

## 💡 How It Works

1. Parent sets time limit (e.g. 10 minutes)
2. App tracks usage of selected apps
3. Once time is used → app gets closed
4. User cannot use the app again for a set period

---

## 🛠️ Tech Stack

* Python
* Tkinter (GUI)
* psutil (Process tracking)

---

## ▶️ How to Run

```bash
pip install psutil
python focuslock.py
```
